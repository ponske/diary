# アセットファイル

このディレクトリには以下のアセットファイルを配置してください：

- `icon.png` - アプリアイコン (1024x1024)
- `splash.png` - スプラッシュスクリーン画像
- `adaptive-icon.png` - Android用アダプティブアイコン
- `favicon.png` - Web用ファビコン

## アイコンの作成

Expo CLIを使用してアイコンを生成することもできます：

```bash
npx expo install @expo/image-utils
npx expo prebuild
```

または、Expoのオンラインツールを使用：
https://docs.expo.dev/guides/app-icons/
